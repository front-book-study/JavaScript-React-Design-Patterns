### 가져오는 방법

- 모듈 객체 : \* from
- 정적 : 최상위에 선언. 코드 실행 전 모듈 다운로드. 성능에 문제 생길 수 있음
- 동적 : 필요한 시점에 로드. 초기 로딩 시간 줄일 수 있음.
- 상호작용시 가져오기
- 화면에 보이면 가져오기 : intersection observer api
