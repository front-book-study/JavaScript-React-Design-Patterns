# 디자인 패턴의 유형

### 디자인 패턴 유형 세 가지

어떤 문제를 해결하냐에 따른 기준으로 분류

- 생성 패턴
- 구조 패턴
- 행위 패턴

## 생성 패턴

주어진 상황에 적합한 객체를 생성하는 방법에 중점

기본적인 객체 생성 방식은 복잡도를 높일 수 있어 이 과정을 제어하여 문제를 해결하고자 함

- 생성자
- 팩토리
- 추상
- 프로토타입
- 싱글톤
- 빌더

## 구조 패턴

객체의 구성과 각 객체 간의 관계를 인식하는 방법에 중점

시스템의 어느 한 부분이 변경되더라도 다른 부분에 영향이 가지 않도록 도와줌

설계 목적에 맞지 않는 부분 개선에 도움

- 데코레이터
- 퍼사드
- 플라이웨이트
- 어댑터
- 프록시

## 행위 패턴

객체 간 커뮤니케이션 개선이나 간소화에 중점

객체 간 공통적인 커뮤니케이션 패턴 감지, 책임 분배

- 이터레이터
- 중재자
- 관찰자
- 방문자
